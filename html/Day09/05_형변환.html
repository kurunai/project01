casting (형변환) : 데이터 타입을 원하는 형식으로 변경하는 기법

  • string => number : Number("str"), parseInt("str"), parseFloat("str")
  • number => Sting : 123.toString(), (123).toString()
  • String => boolean : Boolean("true")
  • boolean => string : String(true)
  • number => boolean : true = 1 / false = 0 : Boolean(1), boolean(0)
  • boolean => number : Number(true), Number(false)
