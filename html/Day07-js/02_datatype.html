변수나 상수에 할당하는 수 있는 값의 종류, 데이터 구분 및 활용을 위해 매우 중요

  • 숫자 : Number : 정수(1,100,0,-5등등), 실수(소수점이 있는 숫자), NaN(Not a Number : 숫자가 아님이라는 값)
  • 문자 : String : ' '," ",` ` 모든 값을 문자로 인식
  • 불리언 : boolean : true/false : 참/거짓
  • 객체 : Object : 1개 이상의 데이터가 특정 구조를 가지고 있는 형태
  • 배열 : Array : [1.2.3 ...] 형태로 이루어진 집합 (순서, 길이 있음)
  • 미정된 값 : undefained : 값이 할당되지 않음
  • 알수없는 값 : null : 무언가 있긴 하지만 파악이 안됨
  • data type 알고 싶을 때 : typeof 변수 or 값
  • 기본자료형(Primative Type) : 값이 메모리에 직접 할당 Number, boolean, String(참조자료형 사용자의 편의성에 의해 기본자료형으로 사용가능.)
  • 참조자료형(Reference Type) : 값이 저장되어 있는 메모리의 주소를 참조하는 형태. 주소값이 할당 String, Object, Array, function, null, undefined
